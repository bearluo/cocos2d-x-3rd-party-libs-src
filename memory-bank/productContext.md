# 产品上下文

## 产品概述
Cocos2d-x 第三方库构建系统，用于为多个平台构建静态库。

## 支持的库
根据项目结构，包含以下第三方库：
- box2d
- bullet
- chipmunk
- curl
- freetype
- glew
- glfw
- glsl_optimizer
- iconv
- jpeg
- lua
- luajit
- openssl
- png
- rapidjson
- sqlite
- tiff
- uv
- webp
- websockets
- zlib

## 平台支持
- iOS
- Mac
- Android
- Linux
- Tizen
- Windows 10 UWP

## 构建模式
- Release 模式（默认）
- Debug 模式
