# 系统模式

## 构建系统模式
基于 Makefile 的构建系统，使用平台特定的配置文件（.ini 文件）。

## 库管理模式
每个库在 `contrib/src/` 目录下有独立的目录，包含：
- `rules.mak`: 构建规则
- 补丁文件（如需要）
- SHA512SUMS: 校验和文件

## 构建流程模式
1. 下载源代码 tarball
2. 验证校验和
3. 解压源代码
4. 应用补丁（如需要）
5. 配置和编译
6. 安装到目标目录

## Memory Bank 模式
使用专门的模式处理不同开发阶段：
- **VAN**: 初始分析和项目设置
- **PLAN**: 详细规划
- **CREATIVE**: 设计探索
- **IMPLEMENT**: 实现
- **QA**: 质量保证
